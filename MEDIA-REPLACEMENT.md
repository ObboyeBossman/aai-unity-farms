# AAI Unity Farms Media Replacement Guide

This document lists all the placeholder images used in the application. When real photos become available, simply update the URLs in `lib/media.ts` to point to the new image paths (e.g., `/images/hero/hero-home.jpg`). No HTML or React components need to be modified.

## Heroes (Recommended size: 1920x1080)
- `heroHome`: Home page hero background
- `heroAbout`: About page hero background
- `heroExport`: Export page hero background
- `heroProducts`: Products page hero background
- `heroContact`: Contact page hero background

## Products (Recommended size: 800x600)
- `freshEggs`: Fresh Eggs product image
- `broilerChicken`: Live Broiler Chickens product image
- `dressedChicken`: Dressed Broiler Chickens product image
- `dayOldChicks`: Day-Old Chicks product image
- `poultryFeed`: Poultry feed product image
- `ghanaianYam`: Premium Ghanaian Yams product image
- `sweetCorn`: Sweet Corn product image
- `soybeans`: Soybeans product image

## Gallery (Recommended size: 800x600)
- `gallery01` - `gallery06`: General farm and operations photos for the Gallery page

## Team (Recommended size: 400x400)
- `teamMember1` - `teamMember3`: Staff or farmer portraits for the About page

## Blog (Recommended size: 800x600)
- `blogPost1` - `blogPost3`: Article cover images for the Blog page
