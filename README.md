Welcome to the AAI Unity Farms web application. This is a Next.js application tailored for premium poultry and Ghanaian agricultural products.

## Run Locally

**Prerequisites:** Node.js

1. Install dependencies:
   `npm install`
2. Run the app:
   `npm run dev`
